Quantum Circuit Simulator

A lightweight simulator for quantum circuits built from first principles — implementing fundamental gates, statevector simulation, and landmark algorithms including Grover's search and Shor's factoring.

Overview

Most quantum computing education relies on high-level frameworks that abstract away the underlying mathematics. This simulator was built as an exercise in understanding quantum mechanics from the ground up — representing quantum states as complex vectors and gates as unitary matrices, then composing them to build circuits.

The core engine operates on statevectors using tensor product composition for multi-qubit systems. Starting from single-qubit gates (Hadamard, Pauli-X/Y/Z, Phase), the simulator supports entangling gates (CNOT, Toffoli) and full measurement with collapse semantics.

Core Concepts

A quantum state of n qubits is represented as a complex vector of dimension 2ⁿ. Gate application is matrix multiplication:

|ψ'⟩ = U |ψ⟩, where U ∈ U(2ⁿ)

Multi-qubit gates are constructed via Kronecker products. For a single-qubit gate G acting on qubit k of an n-qubit system:

U = I⊗(k) ⊗ G ⊗ I⊗(n−k−1)

Implemented Algorithms

  • Grover's search — quadratic speedup for unstructured database search using amplitude amplification
  • Shor's factoring — exponential speedup for integer factorization via quantum Fourier transform
  • Quantum Teleportation — entanglement-based state transfer protocol
  • Deutsch-Jozsa — exponential oracle query speedup over classical deterministic algorithms
  • Quantum Fourier Transform — the core primitive underlying many quantum speedups
