AI Basketball Referee

An AI-powered system that uses computer vision and deep learning to automatically detect foul calls and violations in basketball games in real time.

Overview

Basketball referees make split-second decisions in high-pressure environments — and they sometimes get it wrong. This project explores whether a trained neural network can replicate (and potentially surpass) human referee judgment by analyzing video frames to detect common violations: traveling, double-dribbling, out-of-bounds, and contact fouls.

The system uses a multi-stage pipeline: object detection for player tracking, pose estimation for body position analysis, and a temporal classifier trained on labeled foul sequences to make final calls.

Key Components

  • Real-time player detection and bounding box tracking via YOLOv8
  • Pose keypoint extraction with MediaPipe for joint angle analysis
  • Temporal sequence classifier using LSTM to capture motion context across frames
  • Custom dataset pipeline for labeling game footage with violation annotations
